# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`
- `GET /api/users/:id`

### Session

- `POST /api/session`
- `DELETE /api/session`
- `GET /api/session`

### Beers

- `GET /api/beers`
- `POST /api/beers`
- `GET /api/beers/:id`
- `PATCH /api/beers/:id`
- `DELETE /api/beers/:id`

### Reviews

- `GET /api/reviews`
- `POST /api/reviews`
- `GET /api/reviews/:id`
- `DELETE /api/reviews/:id`

### Breweries

- `GET /api/breweries`
- `POST /api/breweries`
- `GET /api/breweries/:id`
- `PATCH /api/breweries/:id`
- `DELETE /api/breweries/:id`

### Friendships

- `GET /api/friendships`
- `POST /api/friendships`
- `GET /api/friendships/:id`
- `DELETE /api/friendships/:id`
- `PATCH /api/friendships/:id`
